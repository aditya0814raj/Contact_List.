# **App Name**: Contact Keeper

## Core Features:

- Contact List View: Display a list of contacts, showing relevant information like name and other contact details.
- Contact Search: Implement a search bar to filter contacts by name.
- Add New Contact: Enable users to add new contacts with fields for name, and other details.
- Suggest Contact Info: Suggest the possible contact information from the prompt using AI tool, validating them to ensure only realistic contact info.

## Style Guidelines:

- Primary color: Calm blue (#5DADE2) for a professional yet friendly feel. Chosen for its association with communication and trust.
- Background color: Very light blue (#EBF5FB) to provide a clean and unobtrusive backdrop.
- Accent color: Soft purple (#A569BD) to highlight interactive elements and add a touch of creativity. Its higher saturation contrasts gently with the primary.
- Font: 'PT Sans', a humanist sans-serif suitable for both headlines and body text.
- Use clean, modern icons for actions like 'add', 'search', and to represent contact details like phone number and email.
- Maintain a clean and organized layout, prioritizing readability and ease of navigation.
- Subtle animations for adding or searching contacts, providing a smooth user experience.
